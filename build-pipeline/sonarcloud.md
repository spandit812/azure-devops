
Login to sonar cloud:  
https://sonarcloud.io/login  

Create you account there and login with azure devops account.

1. Import organization by passing organization name and PAT.  
2. Top Right--> My Account-->Security--> Generate Token
3. Go the Azure portal --> Organization--> Marketplace--> Search (Sonar Cloud)--> Get it
4. Go to Azure portal, project setting--> Service connection--> Create service connection--> enter token
